Blog

Topic

I write about all sorts of stuff and in various forms. Do not expect any single topic to be focused.

Language

All my articles are, and in the foreseeable future will be in English. I use British spelling and semi-academic language in all articles, but use American spelling and non-academic language in code.

This website does not track you, include analytics or show advertisement. JavaScript is employed for exactly one purpose: to render $\LaTeX$. If the currently viewed article utilises it, MathJax will be loaded; if not, no JavaScript is loaded.
